## css/canvas动画合集
- 收集css和canvas动画,用vue展示使用结果;欢迎贡献代码。
#### PR格式
- 请使用html格式化后在提交
- css动画在'/animation/css'目录下，canvas动画在'/animation/canvas'
- 不要使用图片
- dome结构如下
```
  animation-name
  ├───README.md // 动画说明
  ├───index.html //文件不超过1000行,可以选择放里面   
  ├───... 
  ├───css
  │   ├─── reset.css
  │   ├───dome.css
  │   ├───...   
  └───js
      ├───dome.js
      ├───...  
```
